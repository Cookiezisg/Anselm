## 总览主页 + 关系图(新人之旅第三站，WRK-072，用户 0718-19 拍板)

`/entities`(或 `/`)**无选中时的默认态 = 总览主页**（`EntitiesOverviewView`，`ui/overview/`；退役「选择实体」墓碑——四海洋默认态就此全闭环：chat=landing / scheduler=Overview / documents=草稿 / entities=Overview）。左岛 rail 顶部加**固定「总览」行**（`entitiesOverviewRowId` 无头置顶，对齐 scheduler 先例；无实体选中即高亮，点它 `context.go('/')`）。页头遵**面包屑律**（用户 0718）：灰 crumb=`Entities`（只到上级）、大标题=`总览`、浮层头只显「总览」两字（非全路径）。三段：

- **五牌**：四主类（Function/Handler/Agent/Workflow）+「配件」（trigger+control+approval **合计**）计数牌（`AnCard` 图标+计数）。恒在、0 显 0（清点是恒常结构）、不可点（rail 就在旁）。计数取 `railModelProvider` 组（与 rail 徽标同源，SSE-live）。
- **关系图（主角，v2「涟漪焦点星图」）**：`AnRelationGraph` 观赏态 framed 预览（720 列内定高、框内平移缩放）。数据源 `relGraphProvider`（`GET /relgraph` 全景快照，autoDispose 每显重取）经 `structuralSubgraph` 过滤到**结构边 equip/link + 结构节点**（溯源 create/edit 边 + 对话节点默认不入场=毛线球放大器）。**默认焦点=最近碰过的实体**（`focusId=mostRecentGraphNodeId(sub.nodes, groups)`：rail 已载行 updatedAt ⋈ 图节点 id），它发柔光·稍大，色彩强度按图距离向外**涟漪衰减**（雾彩色相不变、只降不透明度）。单击节点→进探索态并选中（`/entities/graph?sel=kind:id`），**双击→直进该实体详情页**（`entityLocation`，仅 rail kind），hover 框角「展开↗」→进探索态无选中（`/entities/graph`）。
- **最近更新 5 行**：`recentEntities` 跨 kind 合并 rail 已载行、按 updatedAt desc 取 top5（`AnLedgerRow`：kind 图标+名+vN chip+相对时间），点行进该实体详情。无翻页无「查看全部」（rail 即全部）。

**探索态**（全页 `/entities/graph`，`EntitiesGraphPage`，`ui/graph/`）：**独立无边框 Scaffold 路由，非三岛壳**——宽度对齐 workflow 图编辑器先例（满铺画布 + 透明浮层 chrome + 真可收右岛 `AnIsland`+共享 `rightPanelCollapsedProvider`）。`AnRelationGraph` 满幅（浮缩放条）；涟漪焦点=`selectedId`（URL 选中）否则默认焦点（同总览 `mostRecentGraphNodeId`），hover 临时换焦、双击进实体页；左簇 `AnWindowControls` 预留红绿灯 + 返回 + **「显示溯源」开关**（默认关，开后 `?showProvenance` 纳入 conversation 节点 + create/edit 边=「这场对话生了哪些零件」家谱）；底部**kind 图例即显隐过滤**（一行色点 `_LegendChip`，色点走雾彩板；点击 toggle `hiddenKinds`，纯渲染过滤、布局不变；图例兼职过滤器不另设两件）；右岛**实体卡**（`GraphEntityCard`：kind 色字形+名+vN+描述[rail kind 经 `entityRowFetchProvider` 取]+**关系分组** 装备了/被引用/链接[`relationGroupsFor`，逐行 `AnRefPill` 可点=图内 fly-to `revealId`+`revealToken` 平移居中]+「打开详情→」进实体页）。**Esc 回总览**；选区单向派生自 URL 的 `?sel`。
