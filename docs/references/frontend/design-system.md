# 前端设计系统 + UI 套件（An*）

> demo 的视觉语言（`demo/core/tokens.css` + `core/primitives/*`）以 app 级严谨度移植到 Flutter:
> **设计令牌**（值源）+ **An\* 组件套件**（原语），让 features 纯组装、零 bespoke 样式。
> 本篇随套件 **逐组同提交** 填充（见 §4 路线）。架构/分层见 [`architecture.md`](architecture.md)；决策见 [`ADR 0004`](../../decisions/0004-frontend-flutter-architecture.md)。

## 1. 设计令牌（`core/design`，唯一值源，禁内联 px/hex/ms）

- `tokens.dart` —— 主题无关几何/时间:`AnSpace`(4 网格间距,**值命名**,含 s0)· **语义间距层(角色命名,像 `AnRadius` 那样;widget 代码读这层、不直接读 `AnSpace.sN`——同一种关系全产品一个值,消「有的挤有的松」)**:`AnGap`(元素间:`inlineHair`2/`inline`6[紧凑控件内 icon↔标签]/`inlineLoose`8[行首↔标签]/`stackTight`4/`stack`8/`block`12[**主单位**:块↔块/卡↔卡/段↔段]/`section`24/`region`32)· `AnInset`(面内 padding:`tight`8/4·`snug`12/8·`card`16/12·`island`8·`pageX`24·`pageBottom`48)· `AnFlow`(阅读节奏:`listItem`4/`block`12/`headBody`12·`headBodyTight`8·`headBodyDense`6/`headingTop`24·`subheadingTop`16——**标题上下不对称**,上留多下贴内容)· `AnRadius`(tag/button/chip/card/island/pill)· `AnSize`(行高 32、控件 28、图标 16/12/20、光标 1.5×高 16、就地编辑最小宽 32、就地编辑框内距 editBoxPadX 8/editBoxPadY 4(纵向布局补偿不加行高)、状态图标 40、骨架行 12、步进当前点 18、标签×命中 18、三岛列宽、窗体外廓)· `AnMotion`(fast 120 / mid 240 / slow 340 / breath 1800ms + 打字机 typePerChar 55/deletePerChar 28/typeHold 1400/typeGap 400 + easeOut/spring 缓动)。
- **`AnMotionPref`**(`tokens.dart`,挨着 `AnMotion`)——无障碍动效门控单源:`reduced(c)`/`reducedOrAssistive(c)`(`MediaQuery` aspect 访问器,只在标志翻转 rebuild)。**每个动画 An\* 件 build() 里读它**;装饰循环(shimmer/光标/打字机/breath)门控 `reducedOrAssistive`(屏幕阅读器活跃时持续动效是噪声)。见 §2 动效纪律。
- `colors.dart` —— `AnColors` ThemeExtension(明暗双值 + lerp,糖 `context.colors`)。**中性 chrome + toB 蓝 accent + 功能色**:`accent`=蓝(demo `#0071e3`/暗 `#0a84ff`)——主动作/选中/聚焦/run 状态显蓝;`ok`/`warn`/`danger`(+ soft)语义色 + `skeletonBase`/`skeletonHighlight`(骨架哑底+扫光,单色)。值镜像 demo `tokens.css`。
- **`surfaceSunken`**（`colors.dart` 新增语义面色，明暗双值）—— 比 base `surface` 轻降一档的中性填充，供 contained **非交互**区域（聊天泡 / 凹槽 / 内嵌面板）；明确区别于 `surfaceHover`·`surfaceActive`（hover·选中**状态**色）——拿状态色当静止填充会误示可交互。首用于 Chat 用户泡（`ChatTurn`）。
- `typography.dart` —— `AnText`,模数字阶锚 13px 正文。**UI=随包 Inter VF**(`assets/fonts/InterVariable.ttf`,变量字体 wght 100–900,拉丁/数字字面)+ **中文回落随包 MiSans VF**(`assets/fonts/MiSansVF.ttf`,wght 150–700;fallback 链首位——Inter 本就不含 CJK,两字体皆入包、全平台同字面)+ PingFang SC 兜底。**两档字重铁律**——整套 UI 只用两种字重:正文/标签/次级 **Light(w300)**、一切加粗(强调 / 标题 h1–h3 / 导航 / 标签)**Regular(w400)**;**禁 w500 / w600 / SemiBold**(此前标题跳 w600,在压细 UI 里读起来用力过猛)。**正文 `body` 行高 1.4(密集 UI chrome);阅读列另有 `AnText.reading`(13px w300 · 行高 1.6,给 prose 更多空气,照 Notion 校准)——仅 `AnMarkdown` + 文档阅读面用,32px UI 行不变。** 中央 token **`AnText.bodyWeight`(w300)/ `AnText.emphasisWeight`(w400)**;**加粗一律 `AnText.<style>.weight(AnText.emphasisWeight)`**——`.weight()` 扩展双轴同改(`fontWeight` + `wght` `FontVariation`):base 样式已钉 `wght` 轴,直接 `copyWith(fontWeight:)` 会被轴覆盖、渲染原重;且 `TextField`/`EditableText` 只在显式给轴时才渲对的 VF 字重(否则就地编辑框比展示文字更粗更宽)。层级靠**字号 + 颜色**拉开,不靠更重字重。**代码=JetBrains Mono**(随包,OFL;其语法高亮的加粗是另一套字体、不在两档限内);**mono 回退链 MiSans 置首**——拉丁 mono 全不含 CJK,mono 语境中文(代码注释/工具散文结果/终端输出)须确定性落随包 MiSans,平台 mono 只作尾部保险(排前会不确定遮蔽:测试绑定把未知族解析成 FlutterTest 字体,个别 CJK 渲成实心块)。
- `theme.dart` —— 装配 `ThemeData`,注册 `AnColors` 扩展,剥 Material 涟漪 + **hover/highlight/focus/splash 全置透明**(表面自管态,杜绝 Material 默认灰叠加)。

## 2. 命名 + 纪律

- 组件类 `An<Name>`(`core/ui/`);文件 `an_<name>.dart`。纯框架无关模型在 `core/model/`(无 Flutter import)。
- **颜色/度量只走 token**:widget 内禁裸 hex/rgb/`Color(0x…)`/px(只有 `core/design` 可声明色值)——`make verify` 加 grep 门禁机械兜底（套件落地时接）。
- **悬停/选中底色淡入铁律**:`AnimatedContainer` 静止底用**目标色 alpha=0**——统一走 `Color.whenActive(active)` 扩展(`core/design/colors.dart`),绝不用 `Color(0x00000000)`(transparent black,lerp 到浅色经暗中点=暗闪)。
- **共享判定/令牌(去重单源)**:`Set<WidgetState>.isActive`(hover/press/focus 任一,`an_interactive.dart`)各控件统一取「视觉激活」;`AnOpacity.disabled` 统一禁用变暗;占满宽统一名 `block`(AnButton/AnInput/AnDropdown/AnActionGroup)。
- **交互基座 = `FocusableActionDetector`**(`AnInteractive`,原则 #8):hover/focus 由平台高亮模式驱动(焦点环只在键盘聚焦显)、Enter/Space 走标准 `ActivateIntent`、禁用即不可聚焦、`Semantics(button)` 无障碍。下拉菜单据此 + `FocusScope`/`FocusTraversalGroup` + autofocus 选中行得键盘导航(方向键 + Enter)。
- **文案只走 i18n**:严禁硬编码中英文,走 slang `context.t.<key>`(见 §3)。
- **动效 + 生命周期纪律(动画件铁律,源自 G2 调研 [`WRK-037`](../../archive/g2-feedback-states/README.md)):** ① **每个纯动效都附"成品感"静态兜底**(WCAG 2.2.2/2.3.3)——`AnMotionPref` 门控,reduced 时**不启动**循环(`.repeat()` 默认不被 `disableAnimations` 缩短)并渲染**收尾完成的静止帧**(Skeleton 静态哑底/Typewriter 完整主句/Stepper 瞬切/StatusDot 实心点);② **急切初始化**——controller/ticker/timer 绝不放 `late final =` 字段初始化器(惰性首次读可在 teardown 触发→崩),在 `initState` 赋值;③ **永远 dispose + mounted 守卫**;自重排 Timer 持有于字段、dispose 取消;④ **离屏即静默**——循环视觉走 vsync 控制器(TickerMode 离屏自停),Timer 仅用于离散相位切换;⑤ **单 ticker 默认**(`SingleTickerProviderStateMixin`);⑥ **作用域 rebuild**——`AnimatedBuilder(animation,builder,child)` + 循环叶子裹 `RepaintBoundary`;⑦ **异步状态→`Semantics(liveRegion)`(永远 polite),warn/danger 急迫另发 `SemanticsService.sendAnnouncement(assertive)`**;装饰动效 `ExcludeSemantics`;⑧ **状态不靠色单独**(图标+文字+朗读词,WCAG 1.4.1)。matrix 门禁加 **reduced 轴**(每 specimen 在 `disableAnimations` 下须 `pumpAndSettle` 收敛——忘门控的循环卡死=FAIL)。
- **reduced 门控选择(G3 起明确,源 [`WRK-038`](../../archive/g3-rows-cards/README.md) §7):动效分三类、门控不同。** ① **循环/装饰动效**(shimmer / breath / typewriter / 光标闪烁)走 `reducedOrAssistive`(屏读活跃时持续动效是噪声);② **功能性一次性 reveal**(AnRowDetail 高度揭示、AnRow icon↔chevron 互换+旋转)走 `reduced`(真时 `Duration.zero` 即时到位、状态一致);③ **功能性微反馈/微过渡**(focus 边 / hover 提墨 / press / AnDropdown caret 旋转 / 就地编辑 idle↔editing swap)本就一次性 settle、天然过 matrix reduced 轴,**可不显式门控**(若用 `Animated*` 则 reduced 时取 `Duration.zero`)。**铁律:绝不用 `reducedOrAssistive` 门控功能性反馈 / reveal**(会剥夺屏读用户的 hover/focus/展开线索)——`reducedOrAssistive` 只留给装饰循环。
- 注释 S11 双语 Why-not-What。

## 5. 原语目录（An\*，逐组）

> reference = 当前事实投影。本节只留耐久「是什么 / 关键 props / a11y / 仍咬人的坑」;建造过程（复审轮次、决策推演、演化史）在各组归档建造日志。

### G1 基础控件 · 无独立建造日志(G0/G1 早于归档约定,建造/复审史见 git;后续跨组演化:AnTwoZone 升格见 [WRK-038](../../archive/g3-rows-cards/README.md)、AnPopover delegate 重写 / AnDropdown 行壳标准化见 [WRK-039](../../archive/g4-nav-shell/README.md))
- **AnStatusDot** — 状态指示小圆点。取 `AnStatus/AnTone` 折叠出的语义色(idle/run/wait/err/done → none/accent/warn/danger/ok);reduced 静态兜底=实心点。a11y:状态绝不靠色单独,点本身装饰 `ExcludeSemantics`、状态语义由调用方在 label/meta 复述。
- **AnBadge** — 徽章标签,带 `AnTone` 语义色(经 `core/ui/tone.dart` 的 AnTone→色 映射)。
- **AnGroupLabel** — 分组小标题 / 段标签(大写灰 meta 字色 + emphasis 字重)。`padding` 参数化——`padding=zero` 供 `AnSection` caption 复用同源字色/字重(单源,不另搓)。
- **AnButton** — 通用按钮,搭 `AnInteractive`(hover/focus/press/disabled 态 + Enter/Space 激活 + 禁用不可聚焦 + `Semantics(button)`)。变体 ghost / primary、`confirmTone`(danger/primary)、`iconOnly(sm/md/lg)`;`block` 占满宽(需有界父,无界父优雅退化不崩)。坑:**无 accent 文字变体**——需要 accent 小钮时直接搭 `AnInteractive`,不为单点膨胀按钮。
- **AnInput** — `TextField` 封装,**需 Material 祖先**(app 壳与 gallery 均提供,缺则崩)。seamless 无边变体供就地编辑;`style` 覆写基样式(色仍按 readOnly/enabled)、`tabular` 数字同宽、`onTapOutside` 透传(blur-commit);`block` 占满宽。坑:VF 字重须**显式给 `wght` 轴**(走 `AnText.<style>.weight(...)`,勿裸 `copyWith(fontWeight:)`),否则 `TextField/EditableText` 渲原重 → 编辑框比展示文字更粗更宽。
- **AnComposer** — 聊天发送框 chrome 原语(`an_composer.dart`,Phase 4.2 新增;= demo `an-composer` 演变输入)。单行=全圆**药丸**、盒高于一行(文字换行 **或** 附件条)→**卡片**圆角;文字换行时**钮组 reflow**(单行 `[lead·edit·tail]` → edit 占整行 + lead/trailing 落下排,ChatGPT 式)。只拥有视觉壳 + 类型化 `TextField`(注入 `controller`/`focusNode`)+ 聚焦光环;**所有行为**(send/stop、@提及、附件、键)留宿主,经 `lead`(左动作)/`trailing`(send↔stop morph)/`attachments`(输入上方条)注入;`floating`=landing 浮起阴影(`shadowFloat`)。**动效(每个转移都动、无硬切、reduced 即时)**:形变 + 一切高度变化(reflow / 附件条)走**一个 `AnimatedSize`** + 半径 `AnimatedContainer`,`slow`/`spring` 同时——圆角随盒长大;`trailing` 走 **`AnimatedSwitcher`**(scale 0.8→1 + fade,`mid`)——send 首键出现、生成时换 stop 一处搞定(**宿主须给 trailing 加 `ValueKey('send')`/`('stop')`** 才交叉淡、不硬切);**聚焦光环解耦**(`fast` 淡入,反馈不骑 slow 形变)。**光环=三层**:柔光 `accentSoft` 在盒**背后**(内部被白底盖住、只露外圈、**无蓝内填**)· `accentLine` 描边在盒**上面**(仅边缘)· 中间白盒;基础边恒中性 `line`(主输入面**破例给淡 accent**,异于 AnInput 单色克制)。坑:① 多行判定用 `TextPainter` 数行、对**固定参考宽**(`_singleLineReserve`)算防抖;② **圆角判据 `tall = 换行 ‖ 有附件`**(全药丸只留「单行且上方无物」),reflow 判据仅 `multiline`——两者**解耦**;③ **发送钮显隐是宿主的事**——原语只渲 `trailing`(空 null / send↑ / stop);④ 换行文字左内距从按钮几何**派生** `(control−icon)/2 − hairline`,光学左缘与下排图标像素级贴齐、自愈;⑤ **聚焦辉光必须在盒背后、accent 描边在盒上面**——辉光覆层若放盒**上面**会画到白底之上、透出蓝内填(踩过);⑥ **trailing 必须与 lead 同控件档**——单行高取子件 max,更高档的 send/stop 首键出现会把整盒撑高一档、再被 AnimatedSize 平滑成「突长高」(踩过:lead sm(24) + send 默认 md(28) = 首键 +4px;编辑框行盒本身**无嫌疑**——Material `TextField` 在 `strutStyle:null` 时默认注入 `forceStrutHeight` strut,行高对回落字形恒定);全部 `Animated*` reduced 时 `Duration.zero`;需 Material 祖先(同 AnInput)。
- **AnActionGroup** — 一组按钮的横向编排;`end` 右锚(内部包 `SizedBox(width:infinity)`)、`block` 占满宽。坑:**在已右锚的容器(`AnTwoZone`)内或裸 `Row` 内绝不传 `end:true`**——无限宽会撑崩 Row(模式:外层 AnTwoZone 右锚 + 内层 AnActionGroup 不带 end)。

### G3 行与卡 · 建造日志 [WRK-038](../../archive/g3-rows-cards/README.md)

**共享几何/编辑原语（先于消费件落地）**

- **AnTwoZone** — 右锚两区布局原语（`an_two_zone.dart`，从 AnDropdown private `_TwoZone` 升格）。label 占满左（Expanded）+ meta 上限右（≤45%）+ trailing 任意 Widget 钉右，各自省略。被 AnDropdown / AnSection head / AnInfoCard head / AnRow trail 复用。坑：**键值行（AnKv/AnField/AnEditableValue）不能用它**——它 trailing 是内容宽不缩，而键值行要 value 吃尽余量贴右（动态），几何不同，那些走 AnLeadValue。
- **AnLeadValue** — 动态键值行几何核（`an_lead_value.dart`，`SlottedMultiChildRenderObjectWidget` 自研 RenderBox）。leading 贴内容靠左 + value 吃尽「行宽−leading」真实剩余贴右、撞回 leading 才省略；`afterLeading`（铅笔）贴 leading 右、`afterValue`（✓✕/悬停铅笔）钉最右、**0 宽时不加间距**（静态值真贴右缘、两端对齐；悬停铅笔长出才左移）；`wrap` 时 value 填满左对齐换行。leading 取 `min(内容, 0.6·avail)`（私有 `_leadingMaxFraction` 防 value 饿死，留 ≥0.4 + ✓✕ 永在）；实现 intrinsic 安全、无界宽降级为内容宽不崩。被 AnKv/AnField/AnEditableValue 共用。坑：**这不是 Row、RenderFlex 做不到**（Expanded 按 flex 因子预分配、loose Flexible 余量困自身 slot 不回收；`LayoutBuilder` 在任何 intrinsic pass 下崩；非 flex 封顶 leading 不让位致窄编辑行 ✓✕ 溢出）——别想着改回 Row。
- **AnAutoGrid** — auto-fit 响应式块网格（`an_auto_grid.dart`，= CSS `repeat(auto-fit, minmax(W,1fr))`），HAND-ROLL 编排 `LayoutBuilder`（测宽）+ `Wrap`（每行按内容高）+ `SizedBox`（等宽列）。列数 `n=floor((w+gap)/(minW+gap)).clamp(1, childCount)`、列宽 `colW=floorToDouble((w−(n−1)gap)/n)`（向下取整避 FP 误折行，代价：余 <n px 作不可见右边距、非严格 1fr 均摊）；`minColWidth` 默认 `AnSize.block`、gap 默认 `AnSpace.s16`。无界宽 → 单列降级。纯静态、reduced N/A、不加 Semantics 包裹（子件各自可达）。坑：原生 `SliverGridDelegateWithMaxCrossAxisExtent` 不可用——单一 `childAspectRatio` 钉死全表统一行高，违各行按内容定高，别图省事换回它。
- **AnSeamlessField** — 就地编辑字段叶子（AnInlineEdit + AnEditableValue 共用）。`DryIntrinsicWidth`（绕 TextField 嵌套/滚动 dry-layout 断言）+ `caretEndPad` + `inlineEditMin` + seamless `AnInput` + Esc/Enter/onTapOutside；`framed`=编辑态画 1px lineStrong 内描边 + surface 底 + tag 圆角框，**纵向零布局高（`Stack`+`Positioned` 溢出固定行余量）、仅右生长**；选区策略由调用方设（rename 全选 / 改值落末）。坑：不给 placeholder（污染固有宽 flutter#93337）。
- **AnEditableValue** — AnField + AnKv 共用的就地编辑核（`an_editable_value.dart`，公开可单测，= demo field.js）。idle 只读:铅笔**收成 0 宽**故值静态**贴右**（两端对齐）；hover / 键盘聚焦时铅笔**长出 controlSm**（`ClipRect`+`Align(widthFactor)`）、把该行值**挤左**腾位（钮常驻树键盘可达、widthFactor+opacity 揭示）→ 点铅笔值换 AnSeamlessField（或内联 AnDropdown）、**同一最右位置**换成 取消/保存（**单锚、贴其所编之值**，靠近 value 而非 label；即 AnEditAffordance 的同处三连模型）。Enter/✓/失焦提交（blur-commit，`AnInput.onTapOutside`），Esc/✕ 取消；`rowHeight` 参数化（Field islandHead 44 / Kv row 32），展示值镜像 AnInput seamless 样式故切换不跳、**恒单行右省略**（wrap 参数已删——可编辑值钦定贴右，长文本不是就地编辑值的活）；select 行 ghost 下拉同样**贴右填满、不留轨**（下拉本身即常活编辑器、无悬停揭示）；铅笔 a11y 标签=`a11y.editField(field:)`（一列多铅笔可分辨）；行几何走 AnLeadValue（铅笔/取消保存占 `afterValue` 槽）。a11y：进编辑 `SemanticsService.sendAnnouncement`（polite，i18n `a11y.editingField`）；`Semantics(container, explicitChildNodes)` 子件各自可达。坑：① abort 经一次性 `_finished` 守卫优先 + ✓✕ 套 `TextFieldTapRegion`（点 ✓✕ 不算字段外、不误触失焦提交 = 取消优先）；② 焦点回落**按完成来源**——键盘（Enter/Esc）回落铅笔续导航，指针（点 ✓✕/失焦）不回落（否则铅笔被聚焦致 revealPencil 卡可见 + 画焦点框），**不能用 `FocusManager.highlightMode` 判**（桌面鼠标也是 traditional），按调用点判；③ 指针完成须在重建前 `primaryFocus?.unfocus()`（否则 Flutter 把焦点自动恢复到幸存铅笔）。
- **AnExpandReveal** — 共用折叠/展开揭示原语（`core/ui`；建造日志 [WRK-039](../../archive/g4-nav-shell/README.md)）。控制器驱动 `ClipRect` + `Align(heightFactor)`（ExpansionTile 习语），仅向下；open 显 child、否则补间到 0 高**且全收后从树移除**（收起行不可聚焦/不被屏读）；`duration=Duration.zero` 强制即时（过滤强制展开用）。AnRowDetail 用（AnSidebarList 虚拟化重写后改用 `SliverAnimatedList` 的 size-tween 折叠、不再用它）。坑：**非 `AnimatedSize`**——AnimatedSize 在 performLayout 内子件 resize 自脏断言、**嵌套（树枝形态）即崩**，故必用 ClipRect+Align；圆角裁切交外层 ClipRRect/AnIsland。
