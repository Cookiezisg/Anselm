# 前端设计系统 + UI 套件（An*）

> demo 的视觉语言（`demo/core/tokens.css` + `core/primitives/*`）以 app 级严谨度移植到 Flutter:
> **设计令牌**（值源）+ **An\* 组件套件**（原语），让 features 纯组装、零 bespoke 样式。
> 本篇随套件 **逐组同提交** 填充（见 §4 路线）。架构/分层见 [`architecture.md`](architecture.md)；决策见 [`ADR 0004`](../../decisions/0004-frontend-flutter-architecture.md)。

## 1. 设计令牌（`core/design`，唯一值源，禁内联 px/hex/ms）

- `tokens.dart` —— 主题无关几何/时间:`AnSpace`(4 网格间距)· `AnRadius`(tag/button/chip/card/island/pill)· `AnSize`(行高 32、控件 28、图标 16/12/20、光标 1.5×高 16、就地编辑最小宽 32、就地编辑框内距 editBoxPadX 8/editBoxPadY 4(纵向布局补偿不加行高)、状态图标 40、骨架行 12、步进当前点 18、标签×命中 18、三岛列宽、窗体外廓)· `AnMotion`(fast 120 / mid 240 / slow 340 / breath 1800ms + 打字机 typePerChar 55/deletePerChar 28/typeHold 1400/typeGap 400 + easeOut/spring 缓动)。
- **`AnMotionPref`**(`tokens.dart`,挨着 `AnMotion`)——无障碍动效门控单源:`reduced(c)`/`reducedOrAssistive(c)`(`MediaQuery` aspect 访问器,只在标志翻转 rebuild)。**每个动画 An\* 件 build() 里读它**;装饰循环(shimmer/光标/打字机/breath)门控 `reducedOrAssistive`(屏幕阅读器活跃时持续动效是噪声)。见 §2 动效纪律。
- `colors.dart` —— `AnColors` ThemeExtension(明暗双值 + lerp,糖 `context.colors`)。**中性 chrome + toB 蓝 accent + 功能色**:`accent`=蓝(demo `#0071e3`/暗 `#0a84ff`)——主动作/选中/聚焦/run 状态显蓝;`ok`/`warn`/`danger`(+ soft)语义色 + `skeletonBase`/`skeletonHighlight`(骨架哑底+扫光,单色)。值镜像 demo `tokens.css`。
- `typography.dart` —— `AnText`,模数字阶锚 13px 正文。**UI=随包 MiSans VF**(`assets/fonts/MiSansVF.ttf`,变量字体 wght 150–700,Latin+简中,全平台同字面)+ PingFang SC 兜底。**两档字重铁律**——整套 UI 只用两种字重:正文/标签/次级 **Light(w300)**、一切加粗(强调 / 标题 h1–h3 / 导航 / 标签)**Regular(w400)**;**禁 w500 / w600 / SemiBold**(此前标题跳 w600,在压细 UI 里读起来用力过猛)。中央 token **`AnText.bodyWeight`(w300)/ `AnText.emphasisWeight`(w400)**;**加粗一律 `AnText.<style>.weight(AnText.emphasisWeight)`**——`.weight()` 扩展双轴同改(`fontWeight` + `wght` `FontVariation`):base 样式已钉 `wght` 轴,直接 `copyWith(fontWeight:)` 会被轴覆盖、渲染原重;且 `TextField`/`EditableText` 只在显式给轴时才渲对的 VF 字重(否则就地编辑框比展示文字更粗更宽)。层级靠**字号 + 颜色**拉开,不靠更重字重。**代码=JetBrains Mono**(随包,OFL;其语法高亮的加粗是另一套字体、不在两档限内)。
- `theme.dart` —— 装配 `ThemeData`,注册 `AnColors` 扩展,剥 Material 涟漪 + **hover/highlight/focus/splash 全置透明**(表面自管态,杜绝 Material 默认灰叠加)。

## 2. 命名 + 纪律

- 组件类 `An<Name>`(`core/ui/`);文件 `an_<name>.dart`。纯框架无关模型在 `core/model/`(无 Flutter import)。
- **颜色/度量只走 token**:widget 内禁裸 hex/rgb/`Color(0x…)`/px(只有 `core/design` 可声明色值)——`make verify` 加 grep 门禁机械兜底（套件落地时接）。
- **悬停/选中底色淡入铁律**:`AnimatedContainer` 静止底用**目标色 alpha=0**——统一走 `Color.whenActive(active)` 扩展(`core/design/colors.dart`),绝不用 `Color(0x00000000)`(transparent black,lerp 到浅色经暗中点=暗闪)。
- **共享判定/令牌(去重单源)**:`Set<WidgetState>.isActive`(hover/press/focus 任一,`an_interactive.dart`)各控件统一取「视觉激活」;`AnOpacity.disabled` 统一禁用变暗;占满宽统一名 `block`(AnButton/AnInput/AnDropdown/AnActionGroup)。
- **交互基座 = `FocusableActionDetector`**(`AnInteractive`,原则 #8):hover/focus 由平台高亮模式驱动(焦点环只在键盘聚焦显)、Enter/Space 走标准 `ActivateIntent`、禁用即不可聚焦、`Semantics(button)` 无障碍。下拉菜单据此 + `FocusScope`/`FocusTraversalGroup` + autofocus 选中行得键盘导航(方向键 + Enter)。
- **文案只走 i18n**:严禁硬编码中英文,走 slang `context.t.<key>`(见 §3)。
- **动效 + 生命周期纪律(动画件铁律,源自 G2 调研 [`WRK-037`](../../archive/g2-feedback-states/README.md)):** ① **每个纯动效都附"成品感"静态兜底**(WCAG 2.2.2/2.3.3)——`AnMotionPref` 门控,reduced 时**不启动**循环(`.repeat()` 默认不被 `disableAnimations` 缩短)并渲染**收尾完成的静止帧**(Skeleton 静态哑底/Typewriter 完整主句/Stepper 瞬切/StatusDot 实心点);② **急切初始化**——controller/ticker/timer 绝不放 `late final =` 字段初始化器(惰性首次读可在 teardown 触发→崩),在 `initState` 赋值;③ **永远 dispose + mounted 守卫**;自重排 Timer 持有于字段、dispose 取消;④ **离屏即静默**——循环视觉走 vsync 控制器(TickerMode 离屏自停),Timer 仅用于离散相位切换;⑤ **单 ticker 默认**(`SingleTickerProviderStateMixin`);⑥ **作用域 rebuild**——`AnimatedBuilder(animation,builder,child)` + 循环叶子裹 `RepaintBoundary`;⑦ **异步状态→`Semantics(liveRegion)`(永远 polite),warn/danger 急迫另发 `SemanticsService.sendAnnouncement(assertive)`**;装饰动效 `ExcludeSemantics`;⑧ **状态不靠色单独**(图标+文字+朗读词,WCAG 1.4.1)。matrix 门禁加 **reduced 轴**(每 specimen 在 `disableAnimations` 下须 `pumpAndSettle` 收敛——忘门控的循环卡死=FAIL)。
- **reduced 门控选择(G3 起明确,源 [`WRK-038`](../../archive/g3-rows-cards/README.md) §7):动效分三类、门控不同。** ① **循环/装饰动效**(shimmer / breath / typewriter / 光标闪烁)走 `reducedOrAssistive`(屏读活跃时持续动效是噪声);② **功能性一次性 reveal**(AnRowDetail 高度揭示、AnRow icon↔chevron 互换+旋转)走 `reduced`(真时 `Duration.zero` 即时到位、状态一致);③ **功能性微反馈/微过渡**(focus 边 / hover 提墨 / press / AnDropdown caret 旋转 / 就地编辑 idle↔editing swap)本就一次性 settle、天然过 matrix reduced 轴,**可不显式门控**(若用 `Animated*` 则 reduced 时取 `Duration.zero`)。**铁律:绝不用 `reducedOrAssistive` 门控功能性反馈 / reveal**(会剥夺屏读用户的 hover/focus/展开线索)——`reducedOrAssistive` 只留给装饰循环。
- 注释 S11 双语 Why-not-What。

## 5. 原语目录（An\*，逐组）

> reference = 当前事实投影。本节只留耐久「是什么 / 关键 props / a11y / 仍咬人的坑」;建造过程（复审轮次、决策推演、演化史）在各组归档建造日志。

### G1 基础控件 · 无独立建造日志(G0/G1 早于归档约定,建造/复审史见 git;后续跨组演化:AnTwoZone 升格见 [WRK-038](../../archive/g3-rows-cards/README.md)、AnPopover delegate 重写 / AnDropdown 行壳标准化见 [WRK-039](../../archive/g4-nav-shell/README.md))
- **AnStatusDot** — 状态指示小圆点。取 `AnStatus/AnTone` 折叠出的语义色(idle/run/wait/err/done → none/accent/warn/danger/ok);reduced 静态兜底=实心点。a11y:状态绝不靠色单独,点本身装饰 `ExcludeSemantics`、状态语义由调用方在 label/meta 复述。
- **AnBadge** — 徽章标签,带 `AnTone` 语义色(经 `core/ui/tone.dart` 的 AnTone→色 映射)。
- **AnGroupLabel** — 分组小标题 / 段标签(大写灰 meta 字色 + emphasis 字重)。`padding` 参数化——`padding=zero` 供 `AnSection` caption 复用同源字色/字重(单源,不另搓)。
- **AnButton** — 通用按钮,搭 `AnInteractive`(hover/focus/press/disabled 态 + Enter/Space 激活 + 禁用不可聚焦 + `Semantics(button)`)。变体 ghost / primary、`confirmTone`(danger/primary)、`iconOnly(sm/md/lg)`;`block` 占满宽(需有界父,无界父优雅退化不崩)。坑:**无 accent 文字变体**——需要 accent 小钮时直接搭 `AnInteractive`,不为单点膨胀按钮。
- **AnInput** — `TextField` 封装,**需 Material 祖先**(app 壳与 gallery 均提供,缺则崩)。seamless 无边变体供就地编辑;`style` 覆写基样式(色仍按 readOnly/enabled)、`tabular` 数字同宽、`onTapOutside` 透传(blur-commit);`block` 占满宽。坑:VF 字重须**显式给 `wght` 轴**(走 `AnText.<style>.weight(...)`,勿裸 `copyWith(fontWeight:)`),否则 `TextField/EditableText` 渲原重 → 编辑框比展示文字更粗更宽。
- **AnComposer** — 聊天发送框 chrome 原语(`an_composer.dart`,Phase 4.2 新增;= demo `an-composer` 演变输入)。单行=全圆**药丸**、盒高于一行(文字换行 **或** 附件条)→**卡片**圆角;文字换行时**钮组 reflow**(单行 `[lead·edit·tail]` → edit 占整行 + lead/trailing 落下排,ChatGPT 式)。只拥有视觉壳 + 类型化 `TextField`(注入 `controller`/`focusNode`)+ 聚焦光环;**所有行为**(send/stop、@提及、附件、键)留宿主,经 `lead`(左动作)/`trailing`(send↔stop morph)/`attachments`(输入上方条)注入;`floating`=landing 浮起阴影(`shadowFloat`)。聚焦光环=`accentLine` 边 + 一圈轻 `accentSoft` 柔光(主输入面**破例给淡 accent**,异于 AnInput 单色克制)。坑:① 多行判定用 `TextPainter` 数行、且**对固定参考宽**(`_singleLineReserve`)算,防判定与它驱动的布局互相抖;② **圆角判据 `tall = 换行 ‖ 有附件`**(全药丸只留「单行且上方无物」),reflow 判据仅 `multiline`(换行)——两者**解耦**(附件+单行文字时输入仍单排、只圆角落卡片);③ **发送钮显隐是宿主的事**——原语只渲 `trailing`(空=null / 有字=send↑ / 生成中=stop);④ 换行文字左内距从按钮几何**派生** `(control−icon)/2 − hairline`,使其光学左缘与下排图标字形像素级贴齐、按钮尺寸重调时自愈;⑤ reflow `AnimatedContainer` 受 `AnMotionPref.reduced` 门控;需 Material 祖先(同 AnInput)。
- **AnActionGroup** — 一组按钮的横向编排;`end` 右锚(内部包 `SizedBox(width:infinity)`)、`block` 占满宽。坑:**在已右锚的容器(`AnTwoZone`)内或裸 `Row` 内绝不传 `end:true`**——无限宽会撑崩 Row(模式:外层 AnTwoZone 右锚 + 内层 AnActionGroup 不带 end)。
- **AnEditAffordance** — 就地编辑 affordance 原语:idle 文字 + 铅笔,点进 editing 换框,编辑控件**三连同处**(铅笔→取消/保存在同一位置,异于 `AnEditableValue` 的左右两锚)。被 `AnInlineEdit` 组合。
- **AnInlineEdit** — 就地重命名(组合 `AnEditAffordance` + seamless `AnInput`,定高行、idle↔editing 切换不跳)。idle 文字 + 铅笔跟字尾(超长省略、铅笔钉右);editing 自适应框随打字增长、撑满行宽后取消/保存钮钉右且框横滚保光标可见;Enter 存 / Esc 弃 / 进编辑全选(`startEditing`,Finder F2 习惯)。增长封顶用 `IntrinsicWidth`(经 `DryIntrinsicWidth` 垫片)按输入框自身渲染树定宽=所见即所得,**非**逐键 `TextPainter` 量宽(后者踩变量字体宽度漂移坑);按钮宽经 Row 的 `Flexible` flex pass 自动让位。`style`+`minHeight` 参数让大字号标题(如 OceanHeader H2 24px)就地改名不跳字号;可选 `onAbort` 回调(Esc/取消时发)供宿主清编辑态、回展示件。坑:① 尾留 `caretEndPad=3` 防行尾末字符被光标压住(flutter#24612);② rename 不给 placeholder 避 hint 宽污染(flutter#93337);③ 光标高 `caretHeight=16` 单行专用(多行用 Flutter 默认),空框 `inlineEditMin=32` 兜底可点。
- **AnDropdown** — 下拉选择器,搭 `AnPopover` 落真菜单(点外/Esc/祖先滚动关)。触发器与菜单行共用 `AnTwoZone` 两区;菜单宽 = 触发器宽夹 `[menuMin 200, menuMax 360]`(**精确匹配触发器宽、不用 IntrinsicWidth**,异于 AnMenu);选中勾在**尾随**(异于 AnMenu 的前导勾);开/关 fade + 自顶微缩放(`AnMotion.fast`)。行壳/面板壳走共享 `AnMenuRow`/`AnMenuSurface`(圆角内缩悬浮药丸,与 AnMenu 单源)。a11y:`FocusScope`/`FocusTraversalGroup` + autofocus 选中行 → 方向键 + Enter 键盘导航;关闭时焦点归还触发器(AnPopover 提供)。
- **AnPopover** — 浮层基座(`OverlayPortal` + `CustomSingleChildLayout`/`SingleChildLayoutDelegate`)。**定位用 delegate 而非 `CompositedTransformFollower`**:先量子件再据子件+屏幕尺寸定位 → 能**翻转**(优先 `alignEnd` 尾对齐、越界翻首对齐)+ **夹取**(按 `pad`=s8 夹入屏内、下方放不下翻上方),纯仿射 follower 做不到;锚矩形经 GlobalKey RenderBox 在 build 读。点外/Esc/祖先滚动即关(滚动时 delegate 取的锚已动、无法不滞后一帧,故关而非错位悬停)。dialog/toast 非锚定、**不经此基座**。坑:① `AnimationController` 必须 `initState` 急切创建(非懒 `late final =`,否则从未开过的浮层在 dispose 才首次访问→崩);② `CurvedAnimation`/scale curve 提为字段并在父控制器**前** dispose(CurvedAnimation 持父监听、不 dispose 即泄漏)。

### G2 反馈态 · 建造日志 [WRK-037](../../archive/g2-feedback-states/README.md)

> 全 hand-roll、零新增包;动画件(Skeleton/Typewriter)遵 §2 动效纪律 + `AnMotionPref` 门控。

- **AnCallout** — 通栏语气提示条(`Stateful`,有状态仅为发 announce)。severity→AnTone:**info 无 tone 成员故 map accent**,ok/warn/danger 同名,无新色 token;可 dismissible(danger/阻塞错误默认不可关)、0–2 个 action(放不下换行、绝不截断正文)、纯串或富 InlineSpan。a11y 两机制:polite `Semantics(liveRegion)`(info/ok)+ warn/danger 额外 `SemanticsService.announce(assertive)`(initState + didUpdateWidget 文案/severity 变时重发);icon `ExcludeSemantics`,close 给真 label+焦点。坑:两套 a11y 机制混为一谈是本组最大 bug 面。
- **AnState** — 空/载/错整块占位(`Center`+`ConstrainedBox`→`Column`:大图标→title→可选 hint→可选按钮;`AnStateKind{empty,loading,error}`)。error **单色 `inkFaint`**(红留 Callout、不引 AnToneColors,severity 词只进 a11y label)+ 'Try again' ghost;loading 仅留给短不定等待(保存/鉴权)用 `CircularProgressIndicator.adaptive`,**内容加载委托 `AnSkeleton`、不转圈**。a11y 一个 `Semantics(container, '标题. 提示')`,装饰图标 excludeFromSemantics。坑:loading/error 用 liveRegion 但 **empty 不用**(否则误报)。可选 `detail`(hint 下第三行,灰小 3 行截断——如启动失败的原始错误明细,不进 a11y 主播报)+ `fatal`(error 字形转 `danger`,**仅 app 致命屏**[启动/workspace 门控];默认 false,内容内错仍单色)。

- **AnTypewriter** — 打字机循环(`Stateful`+`SingleTicker`;一个 controller 跑 type→hold→delete→gap→下一句的归一化相位机,**光标闪是同一 value 的第二信号**——保持 Single、绝不为光标加第二 ticker)。`phrases:List<String>` + `loop`(false=打完停+稳定光标)+ showCaret/光标样式;`Text(maxLines:1, overflow:clip)`,预留 `caretHeight+caretEndPad` 防行重排裁字。a11y 两层:`ExcludeSemantics` 裹动画 Text + 外层 `Semantics(完整短语)` 且 **liveRegion=false**(循环 live 会刷爆屏读)。坑:① **字素安全**——只 `String.characters.take/length`,绝不 substring/codeUnit(否则切碎 emoji/CJK);② 视觉循环走 vsync 控制器(TickerMode 离屏自停)、**不用裸 Timer**(Timer 不感知离屏、仍烧帧);③ 急切初始化;④ reduced 走 `reducedOrAssistive` → 渲完整静态主短语(`phrases.first`)、不启动控制器;光标闪 reuse `AnMotion.breath`。
- **AnTags** — 可编辑标签集(`Stateful`,持内联输入 FocusNode/controller + 键盘焦点索引;`Wrap` of `AnInteractive` 药丸[复用 AnBadge 几何 + 可选 AnStatusDot 健康点] + 尾随 seamless `AnInput` 内联添加,约束 `inlineEditMin`、经 `DryIntrinsicWidth`)。每药丸=健康点+label+自有按钮语义的移除-x;single(状态层强制单值)/multi、editable/readOnly;**重复添加=拒+短暂聚焦已存在药丸**(不静默、不加重复);改动派 whole-list。a11y 每药丸 Semantics 节点、移除-x 自有 `Semantics(button, removeTag(name))`、健康点 `ExcludeSemantics`,`FocusTraversalGroup` 定 Tab 序。坑:① Backspace 删末标签**仅在** `controller.text.isEmpty`+折叠选区 offset 0 时触发(Shortcuts/onKeyEvent 须作用域化、别全局绑);② FocusNode/controller 必 dispose、async 增删前 `mounted` 守卫;new token `tagRemoveHit`(x 最小命中区,避 InputChip 'tap 附近就删')。
- **DryIntrinsicWidth** — `IntrinsicWidth` 垫片(`core/ui/dry_intrinsic_width.dart`),绕开 `TextField` 在嵌套/滚动上下文的 dry-layout 断言,让宽度封顶按输入框自身渲染树定(所见即所得、非逐键 `TextPainter` 量宽)。被 `AnInlineEdit`(G1)/`AnSeamlessField`/`AnTags` 内联添加框共用。坑:其耐久细节主家在 G1(AnInlineEdit 段),G2 仅复用——改它须同时核 G1。

### G3 行与卡 · 建造日志 [WRK-038](../../archive/g3-rows-cards/README.md)

**共享几何/编辑原语（先于消费件落地）**

- **AnTwoZone** — 右锚两区布局原语（`an_two_zone.dart`，从 AnDropdown private `_TwoZone` 升格）。label 占满左（Expanded）+ meta 上限右（≤45%）+ trailing 任意 Widget 钉右，各自省略。被 AnDropdown / AnSection head / AnInfoCard head / AnRow trail 复用。坑：**键值行（AnKv/AnField/AnEditableValue）不能用它**——它 trailing 是内容宽不缩，而键值行要 value 吃尽余量贴右（动态），几何不同，那些走 AnLeadValue。
- **AnLeadValue** — 动态键值行几何核（`an_lead_value.dart`，`SlottedMultiChildRenderObjectWidget` 自研 RenderBox）。leading 贴内容靠左 + value 吃尽「行宽−leading」真实剩余贴右、撞回 leading 才省略；`afterLeading`（铅笔）贴 leading 右、`afterValue`（✓✕）钉最右；`wrap` 时 value 填满左对齐换行。leading 取 `min(内容, 0.6·avail)`（私有 `_leadingMaxFraction` 防 value 饿死，留 ≥0.4 + ✓✕ 永在）；实现 intrinsic 安全、无界宽降级为内容宽不崩。被 AnKv/AnField/AnEditableValue 共用。坑：**这不是 Row、RenderFlex 做不到**（Expanded 按 flex 因子预分配、loose Flexible 余量困自身 slot 不回收；`LayoutBuilder` 在任何 intrinsic pass 下崩；非 flex 封顶 leading 不让位致窄编辑行 ✓✕ 溢出）——别想着改回 Row。
- **AnAutoGrid** — auto-fit 响应式块网格（`an_auto_grid.dart`，= CSS `repeat(auto-fit, minmax(W,1fr))`），HAND-ROLL 编排 `LayoutBuilder`（测宽）+ `Wrap`（每行按内容高）+ `SizedBox`（等宽列）。列数 `n=floor((w+gap)/(minW+gap)).clamp(1, childCount)`、列宽 `colW=floorToDouble((w−(n−1)gap)/n)`（向下取整避 FP 误折行，代价：余 <n px 作不可见右边距、非严格 1fr 均摊）；`minColWidth` 默认 `AnSize.block`、gap 默认 `AnSpace.s16`。无界宽 → 单列降级。纯静态、reduced N/A、不加 Semantics 包裹（子件各自可达）。坑：原生 `SliverGridDelegateWithMaxCrossAxisExtent` 不可用——单一 `childAspectRatio` 钉死全表统一行高，违各行按内容定高，别图省事换回它。
- **AnSeamlessField** — 就地编辑字段叶子（AnInlineEdit + AnEditableValue 共用）。`DryIntrinsicWidth`（绕 TextField 嵌套/滚动 dry-layout 断言）+ `caretEndPad` + `inlineEditMin` + seamless `AnInput` + Esc/Enter/onTapOutside；`framed`=编辑态画 1px lineStrong 内描边 + surface 底 + tag 圆角框，**纵向零布局高（`Stack`+`Positioned` 溢出固定行余量）、仅右生长**；选区策略由调用方设（rename 全选 / 改值落末）。坑：不给 placeholder（污染固有宽 flutter#93337）。
- **AnEditableValue** — AnField + AnKv 共用的双锚就地编辑核（`an_editable_value.dart`，公开可单测，= demo field.js）。idle 只读、hover 时 key 右冒铅笔（常驻树键盘可达、opacity 揭示）→ 点铅笔值换 AnSeamlessField（或内联 AnDropdown）、value 右出 取消/保存（**两锚**，异于 AnEditAffordance 同处三连）。Enter/✓/失焦提交（blur-commit，`AnInput.onTapOutside`），Esc/✕ 取消；`rowHeight` 参数化（Field islandHead 44 / Kv row 32），展示值镜像 AnInput seamless 样式故切换不跳；行几何走 AnLeadValue。a11y：进编辑 `SemanticsService.sendAnnouncement`（polite，i18n `a11y.editingField`）；`Semantics(container, explicitChildNodes)` 子件各自可达。坑：① abort 经一次性 `_finished` 守卫优先 + ✓✕ 套 `TextFieldTapRegion`（点 ✓✕ 不算字段外、不误触失焦提交 = 取消优先）；② 焦点回落**按完成来源**——键盘（Enter/Esc）回落铅笔续导航，指针（点 ✓✕/失焦）不回落（否则铅笔被聚焦致 revealPencil 卡可见 + 画焦点框），**不能用 `FocusManager.highlightMode` 判**（桌面鼠标也是 traditional），按调用点判；③ 指针完成须在重建前 `primaryFocus?.unfocus()`（否则 Flutter 把焦点自动恢复到幸存铅笔）。
- **AnExpandReveal** — 共用折叠/展开揭示原语（`core/ui`；建造日志 [WRK-039](../../archive/g4-nav-shell/README.md)）。控制器驱动 `ClipRect` + `Align(heightFactor)`（ExpansionTile 习语），仅向下；open 显 child、否则补间到 0 高**且全收后从树移除**（收起行不可聚焦/不被屏读）；`duration=Duration.zero` 强制即时（过滤强制展开用）。AnRowDetail 用（AnSidebarList 虚拟化重写后改用 `SliverAnimatedList` 的 size-tween 折叠、不再用它）。坑：**非 `AnimatedSize`**——AnimatedSize 在 performLayout 内子件 resize 自脏断言、**嵌套（树枝形态）即崩**，故必用 ClipRect+Align；圆角裁切交外层 ClipRRect/AnIsland。

**地基增强（随本组落，挂在 G0/G1 件上）**

- 随 G3 落的三处地基增强：`AnColors.accentLine`（hairline 描边色，供 emphatic 行 / accent 卡，比 accentSoft 深）· `AnInteractive.expanded`（透传 `Semantics(expanded:)`，供 collapsible 行 / RowDetail，null=非披露不误报）· `AnGroupLabel.padding` 参数化（供 Section caption 复用同源字色/字重）。

**展示/布局件**

- **AnRefPill** — 行内实体提及药丸。类型图标走 `AnIcons.byKey` 单源（开放集 + 「?」兜底，**不另抄 kind→icon 表**）+ 文案；`kind` = 后端 EntityKind 线缆值（`relation/entitykind.go`，`document` 非 `doc`、含 `control`/`approval`，11 类）；`id` 非空才可点（`AnInteractive` button，派 `{kind,id}` 经装配层转 select、**原语不碰导航**），空 = 纯标注（不可聚焦、键盘穿透）；`ConstrainedBox(maxWidth: block)` + 省略，无界父下也安全。a11y：单节点「{类型}: {名称}」（WCAG 1.4.1），图标整 visual `ExcludeSemantics`；hover 提墨功能性微反馈（`reduced` 门控）。
- **AnSection** — 小节容器。`caption`（大写灰 meta，复用 `AnGroupLabel` padding=zero 单源）/ `plain`（`AnText.strong` 文档级标题）/ `quiet`（小写灰 meta 安静小标 + 最紧凑节奏，紧凑头组如 run 终端 output/trace/nodes 段）三 variant；有 label 或 actions 即渲 head，head 走 AnTwoZone（label 左 + 右锚），`actions` 走 `AnActionGroup`（**不带 `end`**——AnTwoZone 已右锚，`end` 会包 `SizedBox(width:infinity)` 撑无限宽崩 Row）；grid 变体委托 AnAutoGrid。a11y：`Semantics(container, explicitChildNodes)` 子件各自可达，label 为 `header` 节点读原始大小写（视觉大写经 `ExcludeSemantics`）。坑：`header:true` 真机 macOS VoiceOver 验证仍待补（环境无音频）。
- **AnKv** — 紧凑定义列表。key 左 · value 经 AnLeadValue 吃余量贴右（动态、撞 key 才省略）、每行 `AnSize.row`、靠字色（key inkMuted / value inkFaint）+ 留白分层（无横线）；可编辑行（`AnKvRow.editable` + 非空 `onChanged`）经 AnEditableValue（铅笔→框/下拉、失焦提交、取消优先），只读行为单一 merge「label: value」语义节点；`mono`=值等宽 + tabular、`wrap`=长值换行；改一行 → 重建整列经 `onChanged` 派出（**whole-list**，对齐 AnTags 非位置参）。value 文本 + 编辑框**无条件 tabular figures**（同 demo `.v` 常驻，mono 仅切字体族）。AnKv + AnField 同 `an_field.dart`。

- **AnRow** — 核心行。三列 `[lead(icon16) | label 1fr | trail]`，整行一个 `AnInteractive`（hover 态驱动 lead/trail 揭示）；lead = 状态 `dot` 或 icon↔chevron（collapsible hover 换 chevron `_HoverSwap` + `open` `AnimatedRotation` 转 90°，走 spring）；trail = meta 与 hover 揭示的 `actions` 叠**同一右锚**（`_HoverSwap`：`Stack(centerRight)` 双 `AnimatedOpacity` 互换不重排）；`hint`→行变高（islandHead）顶对齐；`selected`→surfaceActive，`emphatic`+selected→accentSoft 底 + 左**实心 accent 条**（满饱和，异于 AnCard 选中边的淡 accentLine）；`depth` 每级 iconLg(20) 缩进；`mono` 等宽 label；`passive` 不可聚焦键盘穿透。a11y：collapsible 透 `expanded` 披露语义（屏读播报折叠/展开）；**键盘 ←/→ 展开归树消费方（AnSidebarList roving）、不塞 AnRow**（避双键盘属主），lead 折叠点击仅鼠标（`GestureDetector`），带 `dot` 行状态由调用方在 label/meta 复述（dot 装饰 ExcludeSemantics）。坑：① **短行内容须 `Stack(alignment: center)`**——RenderStack 默认 topStart 在 minHeight 撑高时把单行顶对齐（旧 bug）；accent 条上下双钉满高、不受 alignment 影响；② `_HoverSwap` 隐藏层须 `IgnorePointer`+`ExcludeSemantics`+`ExcludeFocus`（opacity 0 不挡命中/焦点/屏读，否则不可见 action 钮成焦点陷阱 + ghost 命中）；③ opacity 互换即时（`Opacity` 非 Animated，只 chevron 旋转走动画，demo 明示不入过渡）。
